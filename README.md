# nwm - neo window manager

My X11 window manager made in C

(name definitely not inspired by neovim)

nwm is mix of sxwm and bspwm.

it's minimal like sxwm but also has bsp tilling and bspc like configuration (not as advanced as bspc)

## Configuration
nwm is configured with the help of nwmc (a program) so you can run nwmc commands from any other program or language.

the [default config file](nwm.conf) is in bash
